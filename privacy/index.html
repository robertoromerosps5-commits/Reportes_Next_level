Política de Privacidad

Última actualización: 4 de mayo de 2026 · Aplicación: Asistente de pautas Next Level

1. Información que recopilamos

La aplicación Asistente de pautas Next Level, operada por Pra2media, accede a los siguientes datos a través de la API de Meta:

  • Datos de rendimiento de campañas publicitarias (impresiones, clics, gasto, conversiones)
  • Información de cuentas publicitarias asociadas al Business Manager autorizado
  • Métricas de audiencias segmentadas (edad, género, ubicación geográfica)
  • Datos de conjuntos de anuncios y anuncios individuales

No recopilamos datos personales de usuarios finales ni información sensible fuera del contexto publicitario.

2. Cómo usamos la información

Los datos obtenidos mediante la API de Meta se utilizan exclusivamente para:

  • Análisis de rendimiento: Generación de reportes y dashboards de campañas para los clientes de Pra2media
  • Optimización publicitaria: Identificación de oportunidades de mejora en las campañas
  • Reportes de gestión: Presentación de resultados a los anunciantes

Los datos no se comparten con terceros, no se venden y no se utilizan para fines distintos a los descritos.

3. Retención de datos

Los datos de campañas se procesan en tiempo real para la generación de reportes. No almacenamos datos de la API de Meta en bases de datos propias de forma permanente. Los reportes generados son archivos estáticos que reflejan el estado en el momento del análisis.

4. Seguridad

Implementamos las siguientes medidas de seguridad:

  • Los tokens de acceso se almacenan de forma segura y nunca se exponen públicamente
  • El acceso a la API se realiza con permisos mínimos necesarios (ads_read)
  • No se realizan operaciones de escritura o modificación en las cuentas publicitarias
  • Los reportes publicados no contienen credenciales ni tokens de acceso

5. Derechos del usuario

Los titulares de las cuentas publicitarias tienen derecho a:

  • Revocar el acceso de la aplicación en cualquier momento desde la configuración de Meta Business
  • Solicitar la eliminación de cualquier dato asociado a su cuenta
  • Solicitar información sobre qué datos han sido procesados
